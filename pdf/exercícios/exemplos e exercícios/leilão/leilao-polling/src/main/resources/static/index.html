ℹ️ Como funciona o Polling

A cada 2 segundos, o navegador envia uma requisição HTTP GET /api/leilao ao servidor — mesmo que nada tenha mudado.

Observe o contador "Reqs. Inúteis" crescer enquanto nenhum lance é feito. Esse é o custo do polling.

Abra o DevTools → Network para visualizar as requisições repetidas.
