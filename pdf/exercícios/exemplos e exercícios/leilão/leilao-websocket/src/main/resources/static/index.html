⚡ Como funciona o WebSocket

Uma única conexão TCP é estabelecida no carregamento da página e mantida aberta.

O servidor empurra mensagens aos clientes apenas quando há novidade (novo lance ou timer). Não há requisição periódica.

Abra o DevTools → Network → WS para ver o único handshake e o fluxo de frames em tempo real.
